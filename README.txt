Project Name: Inventory System

Project Description:
Web-based inventory management system to help small businesses organize their product inflow and outflow.
The inventory management system will have different authority level (such as admin, default user, owner) and can make designated groups.

Project Members: Keegan Sepiol (back-end), Nicholas Sardinia (back-end), Gabriell Yap (front-end), Tristen Klute (front-end)

User Stories: 
A customer wants the ability to sort his database based on name, acquisition date, or number of available items so they can monitor supply levels.
